# 🌓 Sabian Translation Room

構造から読むサビアンシンボル。

物語にしない。
意味を盛らない。
上下を作らない。

---

## for

意味付けや使命に疑問を感じる方  
子どもの性質を構造で知りたい方  
感情や思い込みを介さずに理解したい方  

---

## サンプル

作者のヘリオ水星  
〔おとめ座 20度〕  
A caravan of cars headed to the West Coast.

構造：
複数の視点が  
同じ方向へ  
列のまま  
移動している  

翻訳：
思考が単発ではなく、まとまりを持って続いている。

もう一つの目線：
ただの移動に見えていたものが、
いつのまにか明確な行き先を帯びているように感じられる。
